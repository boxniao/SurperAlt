Alt + 网盘智能识别助手 - 脚本功能介绍
📋 脚本概述
这是一个功能强大的油猴脚本，集成了文本选择搜索、链接访问、复制功能和智能网盘识别于一体的工具脚本。通过简单的键盘操作，大大提高网络浏览效率。

🎯 核心功能
1. Alt键选择模式
激活方式：在非输入框区域按住Alt键

视觉反馈：页面右上角显示圆形指示器

选择状态：

🟢 绿色：选择模式已激活

数字：已选择字符数

📁 文件夹图标：检测到网盘链接

🔗 链接图标：检测到普通链接

✓ 对勾：复制成功

✕ 叉号：取消选择

2. 文本处理流程（释放Alt键时）
优先网盘识别：自动检测并直接打开网盘链接

普通链接访问：识别并访问标准URL链接

文本搜索：使用Bing搜索选中的普通文本

3. 网盘智能识别
支持的网盘类型（超过30种）
主流网盘：百度网盘、阿里云盘、腾讯微云、蓝奏云

国内网盘：天翼云盘、移动云盘、迅雷云盘、123云盘、夸克网盘

特色服务：奶牛快传、城通网盘、FlowUs息流、PikPak

国际网盘：Mega、Google云端硬盘

其他服务：Steam商店、各种扩展商店

识别能力
✅ 自动提取网盘链接

✅ 智能识别提取码（密码）

✅ 支持链接自动补全

✅ 从超链接文本中提取密码

✅ 支持特殊字符替换（如"点"→"."）

4. 智能操作
直接打开：网盘链接检测到后直接在新标签页打开，无需确认

密码自动附加：提取码自动添加到URL中

兼容性处理：支持框架路由（Vue/React等）的URL参数处理

5. 自动填写密码功能
当用户打开网盘页面时：

🚀 自动检测密码输入框

🎯 自动填充提取码

📝 触发输入事件（兼容Vue/React）

🔘 可选自动点击提交按钮

🔄 智能重试机制（最多10次，指数退避）

6. 快捷键操作
Alt键：激活/退出选择模式

Alt + C：复制选中文本

鼠标右键：取消选择并退出模式

⚙️ 可配置选项
javascript
// 默认配置（可通过代码修改）
setting_active_in_front: true      // 是否前台打开标签页
setting_text_as_password: false    // 超链接文本作为密码
setting_auto_complete: false       // 自动推导网盘链接
setting_auto_click_btn: true       // 自动点击提交按钮

场景3：复制文本

文本被复制到剪贴板

❓ 常见问题
Q: Alt键和其他软件快捷键冲突？
A: 脚本只在非输入框区域生效，且可防止浏览器菜单激活

Q: 如何禁用自动点击提交按钮？
A: 修改脚本中的 setting_auto_click_btn: false

Q: 支持自定义搜索引擎吗？
A: 当前使用Bing搜索，可修改代码中的 BING_SEARCH_URL

Q: 为什么有时不识别网盘链接？
A: 检查文本格式，确保链接完整，或尝试启用自动推导功能
