Alt+文本选择增强插件
插件概述
这是一个强大的浏览器增强脚本，允许用户通过按住Alt键选择文本来执行多种智能操作，包括搜索、链接访问、网盘识别和文本复制等。插件特别优化了对各类网盘链接的智能识别和自动填充功能。

核心功能
1. 基础文本操作
Alt+选择+释放：自动在新标签页中搜索选中的文本

Alt+选择+C键：在释放Alt键前按C键复制选中文本

Alt+选择+右键：取消当前选择操作

2. 智能链接识别
自动识别URL：选中文本中的链接会自动识别并高亮显示

自动补全协议：自动为不完整的URL添加https://前缀

支持多种格式：

标准URL（https://example.com）

IP地址格式（192.168.1.1）

本地主机（localhost）

简写域名（www.example.com）

4. 网盘高级功能
自动提取密码：从选中文本中智能提取提取码（支持多种格式）

自动补全链接：对不完整的网盘链接进行智能补全

超链接文本作为密码：可将超链接内的文本作为提取码（需在设置中开启）

自动填写密码：访问网盘页面时自动填写提取码

自动点击提交：填写密码后自动点击提交按钮

5. 视觉反馈系统
状态指示器：屏幕右上角显示当前操作状态

🟢：Alt模式激活

数字：选中字符数

📁：检测到网盘链接

🔗：检测到普通链接

✓：复制成功

✕：操作取消

实时通知：操作结果通过悬浮通知提示

6. 智能选择增强
强制文本选择：在Alt模式下强制允许文本选择

选择保护：防止选择过程中意外中断

选择恢复：操作后恢复原始选择状态
